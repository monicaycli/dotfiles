# dotfiles

## Setup
```sh
git clone https://github.com/monicaycli/dotfiles.git
cd dotfiles
chmod +x setup.sh
sh setup.sh
```

## some other manual settings
* iTerms2
	- Profiles > Keys:
		+ ⌘←: Send Hex Codes: 0x01
		+ ⌘→: Send Hex Codes: 0x05
		+ ⌘←Delete: Send Hex Codes: 0x15
		+ ⌥←: Send ^[ b
		+ ⌥→: Send ^[ f
		+ ⌥←Delete: Send Hex Codes: 0x1b 0x08
* vim
	- install Plug
